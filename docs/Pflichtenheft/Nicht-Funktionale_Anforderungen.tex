\section{Nicht-Funktionale Anforderungen}

\item Das Programm muss sich von Abstürzen erholen können.

\item Das Programm wird ausgeliefert in folgenden Sprachen:
	\begin{itemize}
	\item Englisch
	\item Deutsch
	\end{itemize}
\end{itemize}
